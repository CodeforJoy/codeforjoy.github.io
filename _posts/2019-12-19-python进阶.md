## python进阶知识
### 模块库
就像C语言一样，python拥有更加丰富的模块库，在使用第三方库时，会用到的工具pip
#### pip
pip[下载地址](https://pypi.org/project/pip/#files)或者直接下载[pip-19.3.1.tar.gz](https://files.pythonhosted.org/packages/ce/ea/9b445176a65ae4ba22dce1d93e4b5fe182f953df71a145f557cffaffc1bf/pip-19.3.1.tar.gz)   
解压缩到本地后，进入到[setup.py]文件所在目录，执行下面命令  
`python setup.py install`  
安装成功后，即可用[pip]来安装第三方模块了
`pip install aiohttp`  
`pip install jinja2`  
`pip install aiomysql`
